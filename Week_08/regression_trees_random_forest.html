Answer to Task 2b Effect of Tree Depth

Yes, the depth of the regression tree has a clear effect on prediction quality.
The shallow tree (max_depth=3) produces only coarse splits, resulting in broad groups with similar predicted values. This leads to underfitting and lower predictive accuracy.

The deeper tree (max_depth=5) creates more detailed splits using additional variables such as rooms, taxable income, and distance to supermarket. As a result, the model captures more structure in the data and provides more accurate predictions.

In summary, a greater tree depth increases model flexibility and usually improves R-squared, although very deep trees may begin to overfit.

Task 2c R-squared comparison

With all predictors included (area, rooms, pop_dens, mean_taxable_income, dist_supermarket), the regression tree achieves an R-squared of 0.5481. After removing area and rooms, the R-squared drops drastically to 0.0349.

This large decrease occurs because area and rooms are the strongest predictors of rental price. Once they are removed, the remaining variables contain only weak explanatory power, so the model can no longer capture the variation in rental prices. As a result, the regression tree performs poorly and the R-squared value collapses.

Task 2f Does the feature importance change after removing the variable "area"? Why?

Yes, the feature importance changes after removing area from the model. Previously, area was by far the most important predictor. Once it is removed, the Random Forest redistributes the importance across the remaining variables.

The variable rooms becomes the most important feature because it partially captures the information that was previously explained by area. The other variables also increase in relative importance, since the model must rely entirely on them to predict rental prices.

This behavior is expected: when the strongest explanatory variable is removed, the model shifts importance to the remaining predictors that can best compensate for the missing information.
